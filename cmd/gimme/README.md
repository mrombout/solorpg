# Gimme

Gimme gives you random things.

## Usage

```
$ gimme quest
Rescue the princess from a highly secured castle.
```

```
$ gimme npc
Giant turtle
```

```
$ gimme place
Glittering carnival
```

```
$ gimme enemy
Zombie
```

```
$ gimme thing
Rusty sword
```

```
$ gimme skill
Heraldry
```
